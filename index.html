About HTML

HTML consists of open tags, closed tags, attributes, and content. It is used to specify the content of web pages.

You can learn more about HTML by visiting the W3Schools site's HTML tutorial.

Here are a few HTML tags that may be useful:

TagWhat It Does
tableMake a table
trTable row
thTable header
olOrdered list
liList item

About CSS

CSS is used for formatting web pages: fonts, colors, layout, etc. There are at least three different ways to specify CSS:

  1. Directly in an element
  2. In a script element in the head
  3. In an external CSS file

The best way is in an external CSS file, so that's what we are going to do first.

You can learn more about CSS by visiting the W3Schools site's CSS tutorial.
